How to use


Priorities

Low – Not urgent
Medium – Normal priority
High – Needs attention

Managing Tasks

  • Click Create to add a new task
  • Use the pencil icon to edit an existing task
  • Use the trash icon to delete (archive) a task
  • Click the status button to cycle through: open → in progress → done
  • When a task is set to done it will automatically be moved into the archive

Date Handling

When creating or editing a task, enter a date. The day of the week (e.g. MON, TUE) will be shown automatically based on the selected date.
